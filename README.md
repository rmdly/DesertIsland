# Desert Island

My first C# project.

A non-vibecoded, text-based island survival game.

You wash up on a deserted island with nothing to your name. You'll meet a few characters along the way, and to make it home you'll need to eat, sleep, scavenge what you can, and find a way to escape... if the island lets you.

## Running it

```bash
dotnet run
```

*(Requires the .NET 8 SDK.)*
